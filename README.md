# 中道改革連合 政策提案プラットフォーム

市民の声を政策に。誰でも参加できるオープンな政策提案の場です。

## このプラットフォームについて

中道改革連合を支援する市民が、自由に政策提案を行い、議論を通じてブラッシュアップし、優れた提案を議員に届けて実現を目指す取り組みです。

## 参加方法

### 1. 政策を提案する

- [新しい政策を提案する（Issue作成）](../../issues/new?template=policy-proposal.yml)
- テンプレートに沿って記入するだけでOKです
- GitHubアカウントが必要です（無料で作成できます）

### 2. 議論に参加する

- [Discussions](../../discussions) でアイデア段階の議論ができます
- 既存の提案にコメントして改善に貢献できます
- 提案への賛同は :+1: リアクションで表明できます

### 3. Discordで話す

- [Discord サーバーに参加する](https://discord.gg/3zGugJHNX5)
- リアルタイムで政策について話し合えます
- GitHubの操作がわからない場合もDiscordでサポートします

## 提案から実現までの流れ

提案の進行状況は [Projects ボード](https://github.com/users/kkhassou/projects/2) で一覧できます。

```
1. 提案（Issue作成）                    → ボード「アイデア」
   ↓
2. コミュニティ議論（コメント・👍投票）   → ボード「検討中」
   ↓
3. 提案書の作成（ブランチ → PR）         → ボード「精査中」
   ↓
4. レビュー・改善（PR上で議論）
   ↓
5. マージ → 提出準備                    → ボード「議員提出準備」
   ↓
6. 議員への提出                         → ボード「提出済」
   ↓
7. 実現                                → ボード「実現済」
```

### 提案の進め方（詳細）

- **Issue**: 提案のホームです。議論・投票はここで行います
- **Pull Request**: 賛同が集まった提案について、提案書を作成・レビューします
- **Projects ボード**: すべての提案の現在のステータスが一目でわかります

詳しくは [CONTRIBUTING.md](CONTRIBUTING.md) をご覧ください。

## 提案のルール

- 建設的で具体的な提案をお願いします
- 個人攻撃や誹謗中傷は禁止です
- 特定政党への攻撃ではなく、政策の中身で議論しましょう
- 詳しくは [行動規範（Code of Conduct）](CODE_OF_CONDUCT.md) をご覧ください

## 提案の書き方

提案の書き方やGitHubの使い方は [CONTRIBUTING.md](CONTRIBUTING.md) をご覧ください。
