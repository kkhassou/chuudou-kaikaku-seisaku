# 中道改革連合 政策提案プラットフォーム

市民の声を政策に。誰でも参加できるオープンな政策提案の場です。

## このプラットフォームについて

中道改革連合を支援する市民が、自由に政策提案を行い、議論を通じてブラッシュアップし、優れた提案を議員に届けて実現を目指す取り組みです。

## 参加方法

### 1. 政策を提案する

- [新しい政策を提案する（Issue作成）](../../issues/new?template=policy-proposal.yml)
- テンプレートに沿って記入するだけでOKです
- GitHubアカウントが必要です（無料で作成できます）

### 2. 議論に参加する

- [Discussions](../../discussions) でアイデア段階の議論ができます
- 既存の提案にコメントして改善に貢献できます
- 提案への賛同は :+1: リアクションで表明できます

### 3. Discordで話す

- [Discord サーバーに参加する](https://discord.gg/3zGugJHNX5)
- リアルタイムで政策について話し合えます
- GitHubの操作がわからない場合もDiscordでサポートします

## 政策分野

| 分野 | 説明 |
|------|------|
| [経済・財政](proposals/economy/) | 税制、雇用、産業政策など |
| [教育・子育て](proposals/education/) | 学校教育、子育て支援など |
| [医療・福祉](proposals/healthcare/) | 医療制度、社会保障など |
| [デジタル・行政改革](proposals/digital/) | DX推進、行政手続きの簡素化など |
| [党内運営](proposals/party-operations/) | 組織運営、活動方針、広報戦略など |
| [その他](proposals/other/) | 上記に当てはまらない政策 |

## 提案から実現までの流れ

```
1. 提案（Issue作成）
   ↓
2. コミュニティ議論（コメント・Discussions）
   ↓
3. 賛同の集約（👍 リアクション）
   ↓
4. ブラッシュアップ（有志による精査・データ補強）
   ↓
5. 議員への提出（採択された提案をレポートとして提出）
   ↓
6. フィードバック（検討結果の共有）
```

## 提案のルール

- 建設的で具体的な提案をお願いします
- 個人攻撃や誹謗中傷は禁止です
- 特定政党への攻撃ではなく、政策の中身で議論しましょう
- 詳しくは [行動規範（Code of Conduct）](CODE_OF_CONDUCT.md) をご覧ください

## 提案の書き方

提案の書き方やGitHubの使い方は [CONTRIBUTING.md](CONTRIBUTING.md) をご覧ください。
